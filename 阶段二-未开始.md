# 第二阶段研发计划

## Cognitive Runtime Foundation

这一阶段的目标不是“让角色更聪明”。

而是：

> 建立生命从“感知世界”到“形成认知”的底层演化机制。

第一阶段已经让世界能够：

* 存在
* 变化
* 反应
* 相互作用

第二阶段要让个体开始：

* 感知
* 记忆
* 关联
* 发现规律
* 形成原始知识

最终为后面的：

* 语言
* 协作
* 仪式
* 工具
* 文明

提供真正的“自然起源”。

---

# 第二阶段总体目标

```text id="y72j0l"
环境变化
→ 感知
→ 注意力竞争
→ 行为
→ 结果
→ 记忆
→ 关联
→ 规律发现
→ 群体传播
→ 文明萌芽
```


# Phase 2-3：Memory Runtime

这是文明的真正种子。

---

# 目标

建立：

```text id="zv7pfk"
经验 → 可积累认知
```

---

# 建议架构

## 一、短时记忆（STM）

持续：

```text id="v2vj8f"
几秒 ~ 几分钟
```

例如：

* 刚看到猛兽
* 刚闻到腐臭
* 火快熄灭

---

## 二、情景记忆（Episodic）

记录：

```text id="dwd48p"
事件 + 结果
```

例如：

```text id="5a2ls6"
吃红果
→ 生病
```

---

## 三、模式记忆（Pattern）

系统自动寻找：

```text id="i44f8x"
重复关联
```

例如：

```text id="6t2cmt"
夜晚
→ 更危险
```

---

## 四、情绪强化

强烈事件：

* 更难遗忘
* 更容易传播

例如：

* 火灾
* 猛兽袭击
* 死亡

---

# 遗忘系统

必须有。

否则：

系统最终爆炸。

---

建议：

```text id="t92z2g"
Decay Curve
Importance Score
Reinforcement Refresh
```

---

# Phase 2-4：Discovery Runtime

这是文明诞生点。

---

# 目标

不是：

```text id="7f6zkf"
“解锁科技”
```

而是：

```text id="h6j73z"
“发现规律”
```

---

# 机制

## 规律检测

当：

```text id="tb5yhl"
事件 A
反复导致
结果 B
```

系统形成：

```text id="f9rddo"
Hypothesis
```

---

# 例子

```text id="4q0z5s"
火 + 肉
→ 更少疾病
```

并不会出现：

> 已解锁烹饪

而是：

* 个体偏好增加
* 模仿增加
* 行为传播增加

---

# 建议结构

```text id="0j7wyj"
DiscoveryEngine
 ├─ CorrelationDetector
 ├─ PatternStrengthener
 ├─ HypothesisGenerator
 └─ ConfidenceTracker
```
